Development
===========
.. index:: Development
OpenTSDB has a long history of excellent contributions from around the globe. If you are interested in helping, by adding new features, fixing bugs, adding tools or simply updating documentation, please read the guidelines below. Then please sign the `contributors agreement <https://docs.google.com/spreadsheet/embeddedform?formkey=dFNiOFROLXJBbFBmMkQtb1hNMWhUUnc6MQ>` and send us a pull request.

Git Repository
^^^^^^^^^^^^^^

OpenTSDB 3.0 is maintained in `GitHub <https://github.com/OpenTSDB/opentsdb/tree/3.0>`_. 

The `master`, `maintenance` and `next` branches are for 2.x releases.
